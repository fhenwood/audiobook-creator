# Preset Voice Samples for VibeVoice & Chatterbox

This directory contains high-quality voice reference samples for zero-shot voice cloning.

## Recommended Voice Sample Sources

### 1. VCTK Corpus (Free)
- 109 English speakers with various British accents
- Download: https://datashare.ed.ac.uk/handle/10283/2651
- License: Open Data Commons Attribution (free to use)

### 2. LibriSpeech (Free)
- Clean English audiobook recordings
- Download: https://www.openslr.org/12/
- Great for narrator-style voices

### 3. COQUI TTS Samples
- Bundled with XTTS installations
- Located at: `/app/models/tts/xtts-v2/samples/`

## Tips for Best Results

1. **Duration**: 10-30 seconds of speech works best
2. **Quality**: Use clean, noise-free recordings (our ClearerVoice enhancer can help!)
3. **Content**: Include varied speech with natural intonation
4. **Format**: WAV files at 24kHz or 48kHz recommended
